WHERE TO GET FIDOIP PACKAGES

1. For Windows it is already included to fidoip-*.tar tarboll.
See fidoip_win-*.zip in upper directory

For point connection you could use on Windows Kubik point package(without fidoip node management system). This is sepearate development by Evgeniy Maletin (2:5020/2140.140)

http://sourceforge.net/projects/kubik-fido
