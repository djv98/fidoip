NAME
       pktinfo - Gives some Info about a Packet

SYNOPSIS
       pktinfo [-h] [-t] <pktNames>

DESCRIPTION
       pktinfo gives some Info about the Packets.

       This  manual  page  was  written  for the Debian GNU/Linux distribution
       because the original program does not have a manual page.

OPTIONS
       -h display msg header information (from/to/subject)
       -t display msg text
       <pktNames> Packets to process
